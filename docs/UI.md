# UI 設計

## 1. ランディングページ

- **ヒーローセクション**
  - keepITのロゴとキャッチコピー
  - 主要機能の概要説明（学習記録、教材管理、進捗可視化）
  - 「今すぐ始める」CTAボタン
- **機能紹介セクション**
  - 学習記録機能の説明とスクリーンショット
  - 教材管理機能の説明とスクリーンショット
  - 進捗可視化機能の説明とスクリーンショット
- **フッター**
  - 利用規約、プライバシーポリシーへのリンク
  - お問い合わせ情報

## 2. 認証・ユーザー管理画面

### 2.1 ログイン画面

- **レイアウト**: 中央配置のカード形式
- **要素**:
  - keepITロゴ
  - 「Googleでログイン」ボタン（Google OIDC認証）
  - 「メールアドレスでログイン」リンク
  - 「新規登録はこちら」リンク
  - 「パスワードを忘れた方」リンク

### 2.2 新規登録画面

- **レイアウト**: ステップ形式（3ステップ）
- **ステップ1**: 基本情報入力
  - メールアドレス入力フィールド
  - パスワード入力フィールド（強度表示）
  - パスワード確認フィールド
- **ステップ2**: プロフィール設定
  - ニックネーム入力
  - 学習目標選択（受験、資格、趣味等）
  - 志望校/資格名入力（任意）
- **ステップ3**: 確認・完了
  - 入力内容の確認
  - 利用規約・プライバシーポリシーへの同意チェック
  - 登録完了メッセージ

### 2.3 パスワードリセット画面

- **メールアドレス入力画面**
- **リセットメール送信完了画面**
- **新パスワード設定画面**

## 3. メインアプリケーション画面

### 3.1 ダッシュボード（ホーム画面）

- **ヘッダー**
  - keepITロゴ
  - ユーザーアバター（ドロップダウンメニュー）
  - 通知アイコン
- **今日の学習状況カード**
  - 今日の学習時間表示
  - 連続学習日数
- **最近の学習記録**
  - 直近3件の学習セッション一覧
  - 教材名、学習時間、日時を表示
- **週間サマリー**
  - 過去7日間の学習時間グラフ
  - 日別の学習時間表示

### 3.2 学習記録画面（タイマー画面）

- **学習セッション設定**
  - 教材選択ドロップダウン
  - 学習目標時間設定（任意）
  - メモ入力フィールド
- **タイマー表示**
  - 大きな円形プログレスバー
  - 経過時間表示（HH:MM:SS形式）
  - 残り時間表示（目標時間設定時）
- **コントロールボタン**
  - 開始/一時停止/終了ボタン
  - リセットボタン
- **学習中の情報表示**
  - 現在の教材名
  - 今日の累計学習時間
  - 連続学習時間

### 3.3 教材管理画面

- **教材一覧画面**
  - 検索・フィルター機能
  - カテゴリ別タブ（全教科、数学、英語等）
  - 教材カード表示（画像、タイトル、進捗率、最終学習日）
  - 「教材を追加」フローティングアクションボタン
- **教材詳細画面**
  - 教材情報（タイトル、カテゴリ、難易度、総ページ数）
  - 進捗状況（完了ページ数/総ページ数、完了率）
  - 学習履歴（日付、学習時間、学習ページ数）
  - 編集・削除ボタン
- **教材編集画面**
  - 基本情報編集フォーム
  - カテゴリ選択
  - 難易度設定（1-5段階）
  - 総ページ数入力
  - 画像アップロード機能

### 3.4 進捗分析画面

- **期間選択**
  - 日/週/月/年別切り替えタブ
  - カスタム期間選択
- **学習時間グラフ**
  - 折れ線グラフ（日別学習時間）
  - 棒グラフ（週別/月別学習時間）
- **教材別分析**
  - 円グラフ（教材別学習時間割合）
  - 教材別進捗率
- **統計情報**
  - 総学習時間
  - 平均学習時間
  - 最長連続学習日数
  - 学習効率スコア

### 3.5 設定画面

- **プロフィール設定**
  - アバター画像変更
  - ニックネーム変更
  - 学習目標の更新
- **通知設定**
  - 学習リマインダー設定
  - 目標達成通知設定
  - メール通知設定
- **データ管理**
  - データエクスポート機能
  - アカウント削除機能
- **ヘルプ・サポート**
  - よくある質問
  - お問い合わせフォーム
  - アプリバージョン情報

## 4. 共通UI要素

### 4.1 ナビゲーション

- **ボトムナビゲーション**（モバイル）
  - ホーム、学習記録、教材、進捗、設定
- **サイドバーナビゲーション**（デスクトップ）
  - ロゴ、メニュー項目、ユーザー情報
- **ヘッダーナビゲーション**
  - ページタイトル、戻るボタン、アクションボタン

### 4.2 共通コンポーネント

- **カードコンポーネント**
  - 統一された角丸、影、余白
- **ボタンコンポーネント**
  - プライマリ、セカンダリ、アウトライン
- **フォームコンポーネント**
  - 入力フィールド、セレクトボックス、チェックボックス
- **モーダルコンポーネント**
  - 確認ダイアログ、フォーム入力モーダル

### 4.3 フィードバック要素

- **ローディング状態**
  - スピナー、スケルトンローダー
- **エラー表示**
  - エラーメッセージ、バリデーションエラー
- **成功表示**
  - トースト通知、成功メッセージ
- **空状態**
  - データがない場合の表示

## 5. レスポンシブデザイン

### 5.1 ブレークポイント

- **モバイル**: 320px - 768px
- **タブレット**: 768px - 1024px
- **デスクトップ**: 1024px以上

### 5.2 アダプティブレイアウト

- **モバイル**: 1カラム、ボトムナビゲーション
- **タブレット**: 2カラム、サイドバーナビゲーション
- **デスクトップ**: 3カラム、フルサイドバー

## 6. アクセシビリティ

### 6.1 キーボードナビゲーション

- Tabキーでの順次移動
- Enterキーでのアクティベーション
- Escapeキーでのモーダルを閉じる

### 6.2 スクリーンリーダー対応

- 適切なARIAラベル
- セマンティックHTML
- フォーカス管理

### 6.3 カラーコントラスト

- WCAG AA準拠のコントラスト比
- カラーブラインド対応
- ダークモード対応
